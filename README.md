# FUTURE_PE_01

## 🧠 Task 01 – AI Website Copy Generator for Local Businesses

### 📌 Task Overview
This project demonstrates the use of **Prompt Engineering** to generate high-quality website copy for a local business.

- **Track:** Prompt Engineering (PE)  
- **Business Type:** Café / Restaurant  
- **Brand Name:** BrewNest Café  
- **Location:** Hyderabad, Telangana  
- **Tools Used:** Claude AI, Google Docs, GitHub  

---

## 🎯 Objective
To generate structured website content using AI, including:
- Homepage copy  
- Service/Menu descriptions  
- Call-To-Action (CTA) sections  

---

## 🧠 Prompt Engineering Strategy
Each prompt was designed using:
- Role Assignment (AI as copywriter)  
- Business Context  
- Target Audience (22–38, professionals & students)  
- Tone Guidance (warm, modern, inviting)  
- Output Structure  

This ensured **consistent and high-quality outputs** across all sections.
